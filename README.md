# zapret-discord-youtube-roblox
# Роблокс официально заблокирован в России.
## Есть 2 способа:

## Первый способ:
### Для того что бы роблокс снова работал, вам нужны:
### [Zapret](https://github.com/Flowseal/zapret-discord-youtube/releases) одной из последних версий
### [Cloudflare WARP](https://one.one.one.one/)

### В запрете заходим в папку lists, открываем файл list-general.txt
### Добавляем туда новые ссылки:
```
roblox.com
css.rbxcdn.com
js.rbxcdn.com
ecsv2.roblox.com
metrics.roblox.com
apis.roblox.com
realtime-signalr.roblox.com
games.roblox.com
tr.rbxcdn.com
thumbnails.roblox.com
```
### Сохраняем
### выключаем запрет если он у вас включен
### Включаем WARP
### если выдало ошибку о регистрации аккаунта или типо того - перезапустите варп
### Заходим в роблокс и играем.


## Второй способ:
### Для того, что бы роблокс снова работал, вам нужны:
### [Zapret](https://github.com/Flowseal/zapret-discord-youtube/releases) одной из последних версий
### Данные из файлов ipset-all.txt и list-general.txt
### Вставляем новые айпишкики в \lists\ipset-all.txt
### Заменяем ссылки в \lists\list-general.txt
### Запускаем service.bat и меняем 6. Switch Game Filter (disabled) на состояние 6. Switch Game Filter (enabled)
### Запускаем general (ALT8).bat
### Заходим в роблокс и играем.


### Не уверен про остальные дженералы, работает на 8 и вроде 6, остальные не проверял

### Так же добавлю готовые архивы для этих способов.
